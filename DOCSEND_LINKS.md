## How to Get Your API Key

1. **Log into DocSend**: https://www.docsend.com/login
2. **Go to Settings** (usually in your account/profile menu)
3. **Look for "API" or "Integrations" or "Developer" section**
4. **Generate or copy your API Key**

## Alternative: Check DocSend Help

If you can't find the API settings:
- **DocSend Help Center**: https://help.docsend.com/
- **Contact DocSend Support**: They can help you find or generate an API key

## Note

DocSend's API access might require:
- A paid plan
- Enterprise account
- Or specific permissions

If you don't see API options, you may need to contact DocSend support to enable API access.
